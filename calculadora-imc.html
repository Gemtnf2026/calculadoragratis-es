¿Qué es el IMC?

El IMC es una medida que relaciona el peso y la altura para estimar si una persona tiene un peso saludable.

IMC = peso / altura²

Clasificación

  • < 18.5 → Bajo peso
  • 18.5 - 24.9 → Normal
  • 25 - 29.9 → Sobrepeso
  • 30+ → Obesidad
